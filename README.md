Project Title

AI Uses by Students – Data Analysis Using SQL

Objective

This project analyzes how students are using Artificial Intelligence tools in their academic and personal work using SQL-based data analysis.

Problem Statement

To understand:

How students use AI tools in daily life
Most popular AI tools among students
Purpose of AI usage (study, coding, assignments, etc.)
Usage frequency patterns

Domain

Education Technology (EdTech)
Data Analytics
Student Behavior Analysis

Tools Used
SQL 🗄️
MySQL / SQLite

📊 Key SQL Concepts Used
SELECT Queries
GROUP BY
ORDER BY
COUNT / Aggregations
WHERE Conditions
Subqueries (if used)

Key Insights

Most students use AI for assignments
ChatGPT / AI chatbots are most popular tools
Coding support is a major use case
AI usage is increasing in education field

Dataset Information
Student responses / survey data
Columns may include:
Student_ID
AI_Tool_Used
Purpose
Frequency
Sample Query
SELECT AI_Tool_Used, COUNT(*) AS usage_count
FROM student_data
GROUP BY AI_Tool_Used
ORDER BY usage_count DESC;

Project Outcome

This project helps to:

Understand student AI usage trends
Improve educational AI tools
Analyze behavior using SQL
